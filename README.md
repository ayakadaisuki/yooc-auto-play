# 易班优课(YOOC)自动播放助手
自动播放YOOC课程视频，支持倍速播放，当前视频结束后自动播放下一节（兼容PC和移动端）

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 🚀 **自动播放** | 页面加载后自动开始播放视频，静音绕过浏览器自动播放限制后恢复音量 |
| ⏩ **倍速播放** | 支持 1x / 1.25x / 1.5x / 2x / 3x / 5x 六档倍速，可随时切换 |
| ➡️ **自动下一节** | 当前视频播放完成后，倒计时自动跳转到下一节课 |
| 🛡️ **破解防快进** | 替换 YOOC 原站的 `nodrag()` 机制，只保留学习进度上报，去除防快进检测 |
| 🔒 **倍速防重置** | 持续监测并保持用户选择的倍速，防止被网站脚本重置 |
| 🖱️ **可拖拽面板** | 右上角悬浮控制面板，可拖拽移动，不影响课程页面操作 |
| 🖥️ **兼容双端** | 同时兼容 PC 网页版和移动端网页版 |

## 📦 安装

### 前置条件

需要在浏览器中安装 **Tampermonkey**（油猴）扩展：

| 浏览器 | 安装地址 |
|--------|---------|
| Chrome | [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) |
| Edge | [Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd) |
| Firefox | [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/tampermonkey/) |

### 安装脚本

1. 点击浏览器右上角的 Tampermonkey 图标 → **添加新脚本**
2. 删除编辑器中的默认内容
3. 将 `yooc-auto-play.user.js` 的全部内容粘贴进去
4. 按 `Ctrl + S` 保存
5. 打开 YOOC 课程播放页面，右上角出现悬浮面板即为安装成功

## 🎮 使用方法

### 界面说明

页面右上角会出现一个可拖拽的悬浮面板：

```
┌──────────────────────┐
│     🎓 YOOC 助手      │
│                      │
│ 倍速：[1x][1.5x][2x]  │
│       [1.25x][3x][5x]│
│                      │
│ 自动下一节：  [开关]   │
│                      │
│   ▶ 播放中 | 2x      │
└──────────────────────┘
```

### 操作方式

- **切换倍速**：点击面板中的倍速按钮，当前选中项高亮显示
- **开关自动下一节**：点击右侧滑块开关
- **手动暂停/恢复**：双击视频画面
- **移动面板**：按住面板标题区域拖拽

### 状态提示说明

| 状态 | 含义 |
|------|------|
| `▶ 播放中 \| 2x` | 正在以 2 倍速播放 |
| `⚠ 请点击视频开始播放` | 浏览器拦截了自动播放，需手动点击一次 |
| `✅ 视频播放完成` | 当前视频已播完 |
| `⏩ 3秒后播放下一节...` | 即将自动跳转到下一节课 |
| `🎉 所有课程已播放完毕！` | 全部课程播放结束 |
| `⏸ 已暂停` | 用户手动暂停 |

## ⚙️ 自定义配置

打开脚本文件，修改顶部的 **配置区**：

```javascript
var SPEED_OPTIONS = [1, 1.25, 1.5, 2, 3, 5];   // 可选倍速列表
var DEFAULT_SPEED = 2;                           // 默认倍速
var AUTO_NEXT     = true;                        // 是否自动跳转下一节（true/false）
var DELAY_NEXT    = 5000;                        // 跳转前等待时间（毫秒）
```
